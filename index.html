[ PROJECTS ]
Image Restoration
OCT 2025
Course Project · EE4C03 Statistical DSP, TU Delft
  • Implemented a custom motion blur filter and Wiener filter from scratch in MATLAB; custom blur matched MATLAB's fspecial() benchmark with SSIM difference < 0.005.
  • Derived the MMSE-optimal Wiener solution estimating signal power spectrum from the distorted image alone; achieved SSIM of 0.40 vs. 0.06 for inverse filtering under Gaussian noise (σ² = 10⁻³).
Edge-Aware Image Editing
JUN 2025
Course Project · AP3132 Advanced Digital Image Processing, TU Delft
  • Implemented Local Laplacian Filtering from scratch in MATLAB, building Gaussian/Laplacian pyramid decomposition and a nonlinear remapping function that distinguishes fine-scale detail from strong edges.
  • Benchmarked against MATLAB's built-in locallapfilt() and a bilateral filter; demonstrated comparable edge-preservation quality with full parametric control over detail (α) and edge contrast (β).
Distributed Consensus Algorithms
JUN 2025
Course Project · EE4540 Distributed Signal Processing, TU Delft
  • Implemented Randomized Gossip and PDMM for distributed average and median consensus over a 201-node sensor network in Python, achieving faster convergence with PDMM.
  • Analyzed robustness under simulated sensor and transmission failures, comparing average and median consensus performance.
Real-Time Object Tracker
DEC 2022
Course Project · ECE437 Sensors & Instrumentation, UIUC
  • Implemented real-time light-source tracking at 20 FPS on a Xilinx FPGA system, using OpenCV for brightness-based detection and a motor-driven correction loop.
  • Wrote I2C, SPI, and GPIO finite state machines in Verilog for sensor communication across a CVM300 imaging sensor, LSM303 IMU, and H-bridge motor controller.
  • Built a multithreaded Python pipeline for high-speed FPGA-to-PC image transfer, achieving 30 FPS throughput independent of IMU processing.
EchoRanger
APR 2022
Course Project · ECE420 Embedded Digital Signal Processing, UIUC
  • Implemented a Java-based Android SONAR ranging application based on Graham's chirp-correlation method, computing cross-correlation between transmitted and received signals to estimate distance.
  • Achieved ±0.2 m accuracy at 3 m range; validated by comparing Android output against a Python reference implementation.
